
## 2주차 과제 '자동차 경주게임'
### 게임진행 과정
```bash
- 1) n대의 자동차의 이름을 입력한다.
    - 이름은 5자 이하만 가능하다.
    - 자동차의 이름은 쉼표를 기준으로 구분한다.
    
- 2) 사용자가 입력한 수만큼 자동차는 멈추거나 전진한다.
    - 이때 값이 4이상이면 전진하고 3이하면 멈춘다
    - 한번의 액션(전진/멈춤)을 할 때 마다 결과를 출력한다.
    
- 3) 입력한 수만큼 전진/멈춤하면 가장 많이 전진한 자동차의 이름을 출력한다.
```
